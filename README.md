# 算法竞赛调试助手

一个基于 CustomTkinter 的本地竞赛调试工作台，包含测试数据生成、文本对比、自动对拍、算法模板库和常用小工具。

## 运行

```powershell
.\.venv\Scripts\python.exe main.py
```

如果需要重新安装依赖：

```powershell
pip install -r requirements.txt
```

## 目录

- `main.py`：应用启动入口
- `cp_toolkit/`：GUI、数据生成、文本对比、模板库、对拍引擎和竞赛工具等源码
- `data/templates.json`：用户算法模板
- `workspace/`：运行时输入输出文件，例如 `data.in`、`error.in`、`std.out`、`sol.out`
- `examples/`：示例程序
- `oldversion/`：旧版本备份

## 本次重构重点

- 拆分原来的单文件窗口类，降低 UI 与业务逻辑耦合
- 美化侧边栏、页面标题、配置面板、日志区域和编辑器区域
- 增加页面滑动过渡和按钮回弹反馈，让点击操作更平滑
- 自动对拍改为后台线程 + 队列刷新 UI，避免跨线程直接操作 Tk 控件
- 对拍使用参数列表调用子进程，减少路径包含空格或特殊字符时的问题
- 模板删除、重命名、搜索、复制与保存逻辑更清晰
- 数据生成支持随机种子和是否输出 `T`

## 实用工具

- `复杂度`：按 `N`、测试组数、时限和每秒操作数估算常见复杂度，并反推可承受规模
- `数论`：GCD/LCM、扩展欧几里得、快速幂、模逆、素性测试、质因数分解、欧拉函数、约数统计
- `组合`：组合数、排列数、素数模下组合/排列取模
- `位运算`：二进制/十六进制、popcount、lowbit、最高/下一个 2 的幂和常用写法
